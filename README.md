# 粒粒 Lily Personal Brand Site

一个面向合作机会的个人品牌网站。当前版本已从旧版 `rebuild-preview` 迁移核心内容，并保留新的深绿、米白、浅绿按钮视觉方向。站点是纯静态文件，打开 `index.html` 即可运行，也可通过 GitHub Pages 上线。

## 本地预览

直接在浏览器打开 `index.html`，或用任意静态服务器托管当前目录。

## 编辑个人信息

主要内容在 `index.html`、`styles.css` 和 `main.js`：

- `index.html`：页面文案、小产品入口、资源货架、课程、案例和二维码弹窗
- `styles.css`：当前视觉系统
- `main.js`：移动导航、解决方案切换、合作表单、微信弹窗

## 上线

仓库推送到 `Metely-ETH/Metely-ETH.github.io` 后，可直接用 GitHub Pages 从 `main` 分支根目录发布。

Hero 图和微信二维码在 `assets/` 目录。
